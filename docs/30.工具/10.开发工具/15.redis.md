## 前言

使用`redis`中遇到的奇葩问题，本文持续更新

## `MISCONF Redis is configured to save RDB snapshots, but is currently not able to persist on`

### 翻译

`Redis`被配置为保存数据库快照，但它目前不能持久化到硬盘。用来修改集合数据的命令不能用。请查看Redis日志的详细错误信息。

### 解决方案

``` shell
vi /etc/sysctl.conf

## 添加一个配置
vm.overcommit_memory = 1
## 保存后然后重启
```
